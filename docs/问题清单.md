# 问题清单

## 1. 世界观相关警告导致无法正常生成

- [ ] **game_style 缺失**：出现「game_style缺失，已设置默认值」警告
- [ ] **world_basic_setting 缺失**：出现「world_basic_setting缺失，已设置默认值」警告
- [ ] **protagonist_ability 缺失**：出现「protagonist_ability缺失，已设置默认值」警告
- [ ] 上述缺失导致**世界观无法正常生成**

---

## 2. 配角图无效图片过多

- [ ] 生成配角图时出现**大量无效图片**
- [ ] 需要排查无效图产生原因（生成失败、校验不严、存储/路径问题等）

---

## 3. 配角图内容与分类错误

- [ ] **配角图不是单人图**：应为主角/配角单人立绘，实际出现多人或非单人
- [ ] **场景图被误归类为配角图**：场景/背景图被错误识别或保存为配角图
- [ ] 需要修正：生成规范（单人、角色特写）与分类/筛选逻辑

---

## 4. 生成耗时长与异步问题

- [ ] **整体生成耗时仍然过长**
- [ ] **异步生成疑似异常**：感觉异步没有真正并行或未生效
- [ ] 需要检查：异步任务调度、并发数、等待/阻塞点、超时与重试

---

## 5. 人物颜值与审美判断

- [ ] **无法由机器稳定判断「颜值高」**：提示词或评分标准不足以约束「好看」
- [ ] **生成人物颜值普遍不理想**
- [ ] 需要优化：提示词（外貌描述、风格参考）、审美相关约束或筛选/重生成策略

---

## 6. 人物生成初期大量 404 请求

- [ ] **现象**：一开始人物生成时出现大量类似日志：
  - `127.0.0.1 - - [18/Feb/2026 16:03:58] "GET /initial/main_character/game_1771401661_x80n9w/main_character.png HTTP/1.1" 404 -`
- [ ] **请求路径**：`/initial/main_character/{game_id}/main_character.png` 返回 404
- [ ] **可能原因**：前端或某处在该图片尚未生成完成时就发起请求；静态文件路由/目录配置问题；game_id 或路径不一致
- [ ] **状态**：目前未找到根因，待排查
